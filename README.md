# project: Student Gradebook & Course Manager
i'm Kobra Rahimi

## The project consists of seven classes:
### The Student class enrolls in courses.
### The Course class maintains a list of enrolled student IDs and contains multiple Assessment objects.
### The Assessment class is the parent class of Quiz, Exam, and Project.
### The display_info() and grade_Message() methods are overridden in all child classes.
### The GradeBook class manages the entire application and records students' grades.
### I used encapsulation in the Student class by using getter and setter decorators for the email attribute.
